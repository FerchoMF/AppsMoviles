# Actividad Flutter – Apps Móviles

## Descripción
Aplicación desarrollada en Flutter como actividad académica.
Incluye cambios dinámicos en la interfaz, uso de SnackBar y navegación
entre pantallas utilizando Navigator.

## Funcionalidades
- Cambio dinámico del título de la aplicación
- Cambio de color del AppBar
- Cambio del texto principal
- SnackBar informativo en cada acción
- Navegación entre pantallas
